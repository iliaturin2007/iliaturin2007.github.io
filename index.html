HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») - с его помощью сделаны все сайты в сети internet.
Весь язык состоит из тегов отаечающих за текст, ссылки(гиперссылки), картинки, видео и аудио.
Также в html входит css нужный для оформления сайтов (изменения размера, цвета, фона, расположения), но о нём поговорим позже.

<!DOCTYPE html> <html> <head> <title>Имя Сайта</title> </head> <body> <p>hello world</p> </body> </html>
